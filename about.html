    • CallCenter logfile > 1GB▾
      • Step 1: Login to any of the MOS Callcenter Siebel Servers (wd3010).

        • Login to server using your username and password.
        Switch to the siebel user pbrun css -u siebel.
        Source Siebel environment source /app/siebel/siebsrvr/siebenv.sh.
        obtain the sadmin password from PSAFE.

      • Step 2: Login to Siebel Server Manager

        • srvrmgr /e siebel /g wd3004 /u sadmin /s wd3010 - you will be prompted for the SADMIN password

      • Step 3: Validate that WfProcMgr component (commands are cut and paste)

        • list comp WfProcMgr
        - if the number under the CP_NUM_RUN column is equal to 0, you may restart the component.
        - If the number under the CP_NUM_RUN column is greater than 0, you will need to rerun the list comp command until no tasks are running.

      • Step 4: Restart the WfProcMgr component (commands are cut and paste)

        • shutdown comp WfProcMgr - this command will gracefully shutdown the component when the current task is completed
        start comp WfProcMgr - restarts the component
        list comp WfProcMgr - Verify that the component "CP_DISP_RUN_STATE" is Running or Online
        exit - to exit out of server manager

      • Step 5: Tar and clean the offending log file (replace %source_file.log% with the file from the alert)

        • cd /app/orionlog/callcenter/wd3010/log
        tar -czvf %new_file.tar.gz% %source_file.log%
        delete original source file
        rm -rf %source_file.log%
